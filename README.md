# 使用

Usage: node index.js [options]

Options:
-i, --input <type> 输入文件路径
-o, --output <type> 输出文件路径
-d, --duration <number> 持续时间（秒） (default: 5)
-r, --replaceSource 替换源文件 (default: false)
-h, --help display help for command
